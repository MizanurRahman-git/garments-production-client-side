<h2>Description:</h2>
FEBRIC FASION is a modern, role-based eCommerce platform for fashion and clothing products. Customers can easily browse, register, and place orders through an intuitive web interface.
<hr>

<h2>The platform supports multiple roles:</h2>
Customer: Can explore products, place orders, and track purchases effortlessly.

Manager: Users can apply for a Manager role. Upon Admin approval, Managers can list and manage their own products, including adding new items, updating product details, managing inventory, and handling customer orders.

Admin: Oversees the entire system, including user management, role assignment, product oversight, and order management, ensuring a secure and seamless shopping experience.
<hr>

<h2>Key Features:</h2>
Role-based access control (Customer, Manager, Admin)

Product listing and inventory management

Order approval and tracking system

Responsive design for all devices

Secure authentication and user management
<hr>

<h2>Tech Stack:</h2>
Frontend: React, Tailwind CSS

Routing: React Router

Backend & Database: MongoDB

Hosting: Firebase Hosting

<hr>
FEBRIC FASION provides a well-structured, scalable, and reliable marketplace for fashion enthusiasts, sellers, and administrators, making online clothing shopping and selling seamless and efficient.
